The difference between input button and button?

Similarities :

​ input type=”button” />——— button type=”button”>anything /button>

​ input type=”submit” />——— button type=”submit”>anything /button>

​ input type=”reset” />——— button type=”reset”>anything /button>

Differences:

​ (1) If you use input button, its value just be text. value=”value”,but if you use button,you can set anything as a button,such as an image,a paragraph.

​ (2)Submit button is the default value of button.
